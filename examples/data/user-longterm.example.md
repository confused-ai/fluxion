# Standing facts about you (optional)

Copy this file to `user-longterm.md` in the same folder. The persona example reads it on every run and adds it to the system prompt so the tutor knows you even on a fresh machine before you say anything.

Examples you might write:

- **Name:** …
- **Level:** …
- **Goals:** …
- **Preferences:** short answers / lots of detail / …

Your chat history is also stored in SQLite (`persona-agent-sessions.db`), so past conversations are remembered when you use the same workspace.
